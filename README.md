# Play Project

A development playground project featuring a modern web application built with Next.js.

## Overview

This project serves as an experimental workspace for web development, featuring a responsive frontend application with modern UI components and best practices.

## Features

- Modern web application built with Next.js
- Responsive design with Tailwind CSS
- Component-based architecture
- Clean and maintainable code structure

## Getting Started

### Prerequisites

- Node.js (v18 or higher)
- npm or yarn

### Installation

1. Clone the repository
2. Navigate to the frontend directory:
   ```bash
   cd frontend
   ```
3. Install dependencies:
   ```bash
   npm install
   ```
4. Run the development server:
   ```bash
   npm run dev
   ```
